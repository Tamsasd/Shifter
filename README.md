
# Shifter

> A 3D Puzzle game with prop-hunt like mechanics built in Unity.
> *This project was originally developed for the "Introduction to 3D game development" course at the University of Debrecen (2025/26).*
> Co-author: [Dominik Bozsik](https://github.com/Zobi101)

## Features

* **Multiple characters:** The game contains 14 different characters, each with it's own movement logic and extra features. The class hierarchy makes this list easily expandable.
* **Interactables:** Implemented multiple kinds of switches, which can be easily connected to Map elements, creating an infinite number of possibilities to puzzle creations.
* **Maps:** The game includes 3 different maps (+1 playground map), all with different elements and solutions.
* **Menu** Fully inclusive main, settings, and pause menu, to change between levels, and customize user experience.

## Tech Stack

* **Language:** C#
* **Engine:** Unity 2022+

## How to Run

1. Clone this repository.
2. Download and import the required environment asset packs.
3. Open the project in Unity.
5. Navigate to `Assets/Scenes/` and open the `MainMenu` scene.
6. Press **Play** in the Unity Editor to play the game!
